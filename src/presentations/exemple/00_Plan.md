# Guide Markdown

durée : 5min

## Objectifs

- Découvrir toutes les fonctionnalités de formatage
- Apprendre la syntaxe Markdown complète
- Visualiser les exemples de code
- Intégrer des ressources multimédias
- Maîtriser les listes et tableaux

## Contenu

1. **Formatage de texte** - Gras, italique, code inline, citations
2. **Titres et structures** - Organisation hiérarchique du contenu
3. **Code et syntaxe** - Blocs de code avec coloration syntaxique
4. **Listes et tableaux** - Organisation des données
5. **Médias et liens** - Images, vidéos et ressources externes

---

Cette présentation est conçue pour tester et démontrer l'ensemble des capacités de rendu de l'application.
