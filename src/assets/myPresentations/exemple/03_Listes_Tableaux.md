## Liste avec paragraphes

1. **Planification**

   La phase de planification est cruciale. Elle permet de définir les objectifs, les ressources nécessaires et le calendrier du projet.

   Points clés :

   - Définir le scope
   - Identifier les risques
   - Allouer les ressources

2. **Exécution**

   Durant cette phase, l'équipe développe les fonctionnalités selon le plan établi. La communication et la collaboration sont essentielles.

   Activités principales :

   - Développement
   - Revues de code
   - Tests continus

3. **Livraison**

   La mise en production nécessite une préparation minutieuse et un suivi post-déploiement attentif.

## Résumé

Les **listes** permettent d'organiser l'information de manière claire, tandis que les **tableaux** sont parfaits pour présenter des données structurées et comparatives.
